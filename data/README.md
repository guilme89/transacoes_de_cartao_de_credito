# Pasta `data`

Coloque aqui o arquivo:

```text
creditcard.csv
```

O dataset usado no projeto é o dataset real de transações de cartão de crédito usado no desafio/aula.

Por boa prática de GitHub, o arquivo `.gitignore` está configurado para não versionar arquivos `.csv` dentro desta pasta, pois bases reais costumam ser pesadas.

Para executar o projeto localmente:

```bash
python src/deteccao_anomalias_transacoes.py
```
