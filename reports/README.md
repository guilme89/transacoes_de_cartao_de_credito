# Relatórios

Esta pasta contém relatórios em Markdown gerados ou preparados para o projeto.

Arquivos principais:

- `relatorio_treinamento.md`: resumo dos resultados obtidos após executar o treinamento com o dataset real;
- `insights_melhoria_processo.md`: recomendações para melhorar o processo de detecção de fraudes;
- `tabela_metricas_markdown.md`: tabela de métricas em formato Markdown para reutilização no README.
